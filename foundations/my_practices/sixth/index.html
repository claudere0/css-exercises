Basic Flexbox Properties

CSS Flexbox (short for css flexible box layout module)

CSS Flexbox is short for the CSS Flexible Box Layout module. Flexbox is a layout model for arranging items (horizontally or vertically) within a container, in a flexible and responsive way. Flexbox makes it easy to design a flexible and responsive layout, without using float or positioning.

To create flex container, use display property with flex value: display: flex;

flex-wrap: wrap;
